TRADING OPERATING SYSTEM

交易員身心作業流程

盤前 · 盤中 · 盤後 · 每週 —— 按表操課,不打分數,只打勾
「我專注於執行我的系統」
盤前 身體重開機 · 5-10分鐘
目的是進入冷靜專注,不是興奮
  • 身體重開機橫膈膜呼吸 + 骨盆前後傾,不是拉筋,重點是腹部不要一直縮著
  • 心率下降吸氣4秒,吐氣6-8秒,做幾次即可
  • 念一次宣言「我專注於執行我的系統」
  • 看盤前確認,寫下三件事今天最大風險 / 今天最多接受幾次停損 / 今天停手條件
盤中 防止交感神經失控
  • 每30分鐘身體檢查肩膀放鬆了嗎?下巴放鬆了嗎?舌頭有沒有頂上顎?腹部是不是一直縮著?
  • 每筆成交後只問一件事「我有沒有照計畫?」只有 有/沒有,不評論賺賠
  • 連續停損 → 立刻離開螢幕走路三分鐘,喝水,不分析
  • 大賺 → 也離開螢幕興奮跟恐懼都會降低判斷品質,同樣對待
盤後 比盤中更重要
  • 5分鐘放鬆做治療師教的呼吸,不要急著檢討,先切回副交感神經
  • 先寫「有沒有照計畫」,不是先寫賺多少
進場規則 ✓ / ✗
停損執行 ✓ / ✗
有沒有追價 ✓ / ✗
情緒:害怕/貪心/生氣/後悔/不甘心
  • 結束儀式關掉看盤軟體,做一件完全無關交易的事(散步、遊戲、動畫、做麵包、小說),讓大腦知道「今天交易結束了」
每週一次 30分鐘 · 不是檢討績效
只回答三個問題,一次只改一件事
  • 這週哪一次最冷靜?
  • 這週哪一次最衝動?
  • 如果重來一次,只改一件事,會改什麼?
此頁不儲存勾選狀態,重新整理會重置 · 完整紀錄請見交易筆記文件
